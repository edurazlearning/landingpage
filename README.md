# EduraZ Landing Page

A premium, minimal marketing landing page for **EduraZ** — designed to feel like the same product as the EduraZ Roadmap app.

EduraZ helps self-learners generate a **personalized learning roadmap in ~30 seconds**, turning any goal into a clear plan with **weekly milestones + daily tasks**.

---

## What this landing page is for

This landing page exists to:
- Introduce EduraZ in a clear, high-trust way
- Explain the product value instantly (without long text)
- Drive users into the app using a strong CTA: **“Try the free app”**
- Let people preview the “roadmap experience” before clicking into the full app

The entire page is built to feel **premium, minimal, and consistent** with the EduraZ Roadmap UI: clean typography, lots of whitespace, rounded cards, subtle borders, and smooth micro-interactions.

---

## Key sections

### 1) Sticky Navigation
A simple top bar with:
- **EduraZ** wordmark
- Theme toggle (light/dark)
- Primary CTA button: **Try the free app**

### 2) Hero (Main Message)
Clear headline + subheadline communicating the core promise:
- “Get a personalized roadmap in 30 seconds.”
- “Stop wasting time searching…”

Plus a trust line that makes it obvious:
✅ **EduraZ is free to use (MVP)**

### 3) Interactive Roadmap Preview (Fast)
The landing page includes an embedded preview card that mimics the EduraZ builder experience:
- goal input
- schedule preferences (timeframe, time per day, experience, style, intensity)
- a **“Generate preview”** button

This preview is intentionally lightweight:
- no AI calls
- no waiting
- instant roadmap output

It exists to show value immediately and push users to the real app for the full experience.

### 4) Examples
Clickable examples that auto-fill the preview:
- Learn HTML & CSS
- Learn Python for quant investing
- Neuroscience fundamentals + study habits

### 5) How It Works
A clean 3-step explanation:
1) Describe your goal  
2) Choose your schedule  
3) Get milestones + daily tasks  

### 6) Benefits + Social Proof
Short benefit cards and testimonial placeholders to increase trust and clarity.

### 7) Footer
Simple links and a final CTA back into the app.

---

## Brand & UI principles

This landing page is built to match the EduraZ product identity:
- white background
- near-black text
- soft muted gray secondary text
- thin borders and subtle shadows
- large rounded corners
- minimal, modern spacing
- smooth transitions (no heavy animations)

The goal is for users to feel like:
**landing page → app = one consistent brand**

---

## Primary CTA

The main action on the page is always:
**Try the free app → https://eduraz.com/**

Everything on this landing page supports that conversion.
